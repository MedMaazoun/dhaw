⚡ Dhaw ضو — comment ça marche ?

Carte open source des coupures d'électricité en Tunisie, au niveau des 264 délégations, pensée pour les périodes de canicule et de délestage.

3 sources de données croisées

Annonces STEG (automatiques). Un robot lit toutes les 10 minutes les flux des médias tunisiens (La Presse, Webdo, Business News, Gnet…) qui relaient les communiqués publiés par la STEG sur son site et sa page Facebook, en extrait les horaires et localités, et met la carte à jour.
Signalements citoyens (temps réel). Chaque visiteur peut signaler « coupure » ou « courant rétabli » dans sa délégation. Les signalements sont partagés entre tous les visiteurs et expirent après 4 h.
Satellite NASA VIIRS (vérification). L'onglet « Nuit satellite » affiche les lumières nocturnes réelles (~500 m, chaque nuit vers 01h30) : comparer une nuit récente à la référence permet de repérer les zones restées dans le noir.

Open source

Code libre (MIT) : frontend statique + scraper Python + GitHub Actions + Supabase. Contributions bienvenues — voir le README du dépôt pour déployer votre propre instance gratuitement.
